GIT CHEAT BASICS

Git Basics

Create an empty Git repo in a specified directory. Run with no argument to inialize the current directory as a git repository.

git init <directory>

Clone repo located at onto the local machine. Original repo can be located on the loca.filesytem or on a remote machine via HTTP or SSH.

git clone <repo>

Define author name to be used in all commits in current repo. Devs commonly use --global flag to set config options for current user.

git config user.name <name>

Stage all changes in directory for the next commit. Replace directory with a file to change a specific file.

git add <directory>

Commit the staged snapshot, but instead of launching a text editor, usemessage as the commit message.

git commit -m <"message">

List which files are staged, unstaged and untracked.

git status

Display the entire commit history using the default format. for customization see additional options.

git log

Show unstaged changes between your index and working directory.

git diff
